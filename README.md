# Quantum Finance Notebooks

This repository contains the curated Jupyter notebooks referenced by the review paper on quantum computing for financial transformation.

## Repository structure

- `module_01_optimisation/`
  - `module_01_classical_vs_qaoa_portfolio.ipynb`
- `module_02_pricing/`
  - `module_02_asian_option_qae.ipynb`
- `module_03_risk/`
  - `module_03_single_asset_var_cvar.ipynb`
  - `module_03_multi_asset_var_cvar.ipynb`
- `module_04_qml/`
  - `module_04_case_a_quantum_classification.ipynb`
  - `module_04_case_b_macro_regime_learning.ipynb`

## Scope

The notebooks are organised by the same module labels used in the review paper:

- Module 1: portfolio optimisation and QAOA benchmarking
- Module 2: quantum amplitude estimation for derivative pricing
- Module 3: VaR/CVaR and risk estimation workflows
- Module 4: quantum machine learning case studies

The manuscript itself is not included in this repository. This repository is intended only for the executable notebook layer.
